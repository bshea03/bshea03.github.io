A Universal Machine

Languages: C

Reads in a text file containing a program (written in UMASM, an assembly language specific to the Universal Machine), then executes that program. Made to emulate the behavior of a computer using eight registers, a segmented, word-oriented memory system, a 32-bit program counter, and an I/O device capable of inputting and outputting unsigned, 8-bit characters.

November 2016

Arith

Languages: C

Reads in a PPM image then compresses it. This is done by converting the pixels' color values from RGB to component video, using a discrete cosine transform, then packing the resultant values into a binary keyword. The program then decompresses the image by unpacking the binary keyword and following the inverse procedure.
